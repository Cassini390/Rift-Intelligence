v2.1 Latest
Visual Redesign
  • New sticky navigation bar with glass-blur effect and theme colour glow
  • Theme switcher moved to nav as compact colour dots
  • Added Space Grotesk display font and JetBrains Mono for all stat values
  • Stat card numbers now use monospaced font for precision and readability
  • Match card champion names use display typeface; KDA values use mono
  • Banner is now purely decorative with improved overlay fade
  • Profile card, stat grid, and match cards have more breathing room
  • Release notes section converted from inline styles to proper CSS classes
v2.0
Theme System & Visual Overhaul
  • 5 selectable themes: Shadow Isles, Demacia, Noxus, Freljord, and Ionia
  • Banner gradient and subtitle update to match the selected theme
  • Theme saved to localStorage and restored on next visit
  • Win/Loss text replaced with coloured pill badges
  • Champion icons added to Best/Worst/Most Played highlight cards
  • Removed KDA and Win/Loss trend charts
  • Visual polish: stat card accents, item row cleanup, info banner styling
v1.9
Security & Configuration Improvements
  • API key moved from browser input to .env file — never sent over the network or stored in the browser
  • Added .gitignore to prevent .env and node_modules being accidentally committed
  • CORS restricted to localhost only
  • Server validates region and returns clear errors for misconfiguration
  • Fixed missing CSS variables causing ARAM and Diamond/Emerald badges to display incorrectly
  • Visual polish: info banner, improved Release Notes section, tag input sizing
v1.8
Scoreboard & Layout Improvements
  • Widened page container from 960px to 1200px for more space
  • Scoreboard teams now stack vertically instead of side by side, giving each player row full width
  • Scoreboard grid updated with dedicated columns for keystone rune, vision score, and trinket
  • Item icons in scoreboard increased from 18px to 24px for better readability
  • Trinket now shown in scoreboard alongside the 6 main item slots
  • LP tracker removed as it is not needed for casual play
v1.7
Shadow Isles Theme & Banner
  • Full Shadow Isles colour scheme — deep blacks, toxic greens, eerie teals and purples
  • Added image banner at the top of the page with mist/glow fallback background
  • Banner image is easily configurable — find the BANNER IMAGE comment in index.html and paste your image URL
  • Page title now uses an animated green gradient with a glow effect
  • All accent colours updated from gold to Shadow Isles green and teal
v1.6
Expandable Match Scoreboard
  • Each match card now has a "Full Scoreboard" expand button
  • Clicking it reveals all 10 players split into Blue and Red teams
  • Shows champion icon, summoner spells, summoner name, K/D/A, gold, CS, and item build per player
  • Your row is highlighted with a star prefix
  • Victory/Defeat label per team with colour coding
